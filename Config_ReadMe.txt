This file is for inforamtion for configurating the app.

1.) In CreateUSers.js file add the credentials regarding host,user,password,dbname for MYSQL.
2.) Install Nodejs and open this project in CMD.Type npm install.
*"npm install" will automatically installs all dependecies. for this app.
3.)Dump the mysql file in the mysql database named as "einsite.sql"
4.)In cmd type the command npm start.
    This will start the server.
